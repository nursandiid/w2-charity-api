# Report API Spec

## Get All Reports
Endpoint : GET /api/reports

Query Params :
```json
{
    "start_date": "",
    "end_date": ""
}
```

## Export Report to PDF File
Endpoint : GET /api/reports/pdf

Query Params :
```json
{
    "start_date": "",
    "end_date": ""
}
```

## Export Report to Excel File
Endpoint : GET /api/reports/excel

Query Params :
```json
{
    "start_date": "",
    "end_date": ""
}
```
