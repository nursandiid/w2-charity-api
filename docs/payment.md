# Payment API Spec

## Create Payment
Endpoint : POST /api/donations/:id/payment

Request Body :
```json
{
    "name": "",
    "nominal": "",
    "bank_id": "",
    "note": "",
    "path_image": ""
}
```

## Get Payment
Endpoint : GET /api/donations/:id/payment

Request Body :
```json
{
    "name": "",
    "nominal": "",
    "bank_id": "",
    "note": "",
    "path_image": ""
}
```
